# اسناد برق و کنترل (بدون PLC)

این پوشه شامل مستندات و ابزارهای لازم برای طراحی و اجرای سیستم کنترل مبتنی بر رله است.

## فایل‌ها
- `electrical-spec.md`: مشخصات و قواعد طراحی برق، بدون استفاده از PLC و با تأکید بر ولتاژ صحیح کویل رله‌ها.
- `cross-reference.csv`: جدول تطبیق با PDF ۸۰ صفحه‌ای (قابل‌خواندن در اکسل). برای هر مورد، صفحه، آدرس ترمینال، عنوان عملکرد، و اصلاح پیشنهادی را ثبت کنید.
- `../public/docs/wiring-flow.svg`: دیاگرام فلشی مسیر سیم‌کشی از ابتدا تا انتها برای رفع ابهام.

## روش کار
1. برای هر صفحه از PDF، اطلاعات اجزا و ترمینال‌ها را در `cross-reference.csv` وارد کنید.
2. هر ناسازگاری بین عنوان عملکرد (قطع/وصل) و آدرس ترمینال را ثبت و اصلاح پیشنهادی ارائه کنید.
3. در صورت نیاز به افزودن رله، نوع رله مطابق ولتاژ موجود (۲۴VDC یا ۱۱۰VDC) تعیین و صفحهٔ اعمال اصلاح را قید کنید.
4. برای شفافیت مسیر، از دیاگرام `wiring-flow.svg` استفاده کنید و شماره سیم، رنگ، و مبدا/مقصد ترمینال‌ها را مستند نمایید.

## نکات مهم
- از تزریق ۱۱۰VDC به کویل ۲۴VDC خودداری شود؛ انتخاب رله باید دقیقاً با ولتاژ موجود هم‌خوان باشد.
- ایزولاسیون مدار فرمان و قدرت، و حفاظت‌های لازم (فیوز، دیود هرسی) رعایت گردد.
- پس از تکمیل جدول، یک بازبینی نهایی برای مطابقت کامل با PDF انجام دهید.
